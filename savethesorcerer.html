Save the Sorcerer

The power of magic spells through an immersive game experience.

Introduction

What is this about

Save the Sorcerer is a short first person shooter game which involves magic as a weapon to destroy the enemies. This game is built using Unity 3D and IBM Watson SDK. It combines exciting combat, exploration, casts and spells. The idea behind this concept is to provide a magical experience to the players by giving the power of spell casting.

Approach

How is it achieved

This project makes use of Unity 3D game engine where the entire world is built. Scripts are written to spawn the enemies and animate them according to the player's movement. IBM Watson SDK is used for speech detection which then checks for the player's voice input for spells. Based on the spells the player casts different attack modes are initiated. The current version of the game detects only 3 spells namely - "Ridiculous", "Expeliamus" and "Vingadiyam Leviyosa".
