Game of Hands

A hand gesture game that can be played with computer

Introduction

What is this about

As the name signifies this project is a game of hand gestures. This was inspired from the famous Rock, Paper, Scissor game. The concept here is to create a playful experience which involves some kind of physical interaction with the digital world. This project that makes use of arduino and photo sensors to create an immersive gaming experience that users can play along with the computer. The agenda here is to make gestures that do not match the gestures made by the computer. Based on the gestures made scores are calculated. The ultimate goal is this game is to make high score. This project was done as part of Creativity and Computation course at Parsons School of Design.

Approach

How is it achieved

This project makes use of simple principles of light. A photosensor is used that can detect the presence of visible light and send a signal to the Arduino. A photosensor is placed on the tip of each finger, whenever a particular finger is closed that sensor stops sending signals. Based on these sensing patterns, various gestures are identified and then sent to the computer via microcontroller where it tallies with computer's gesture and provides a result.

Final product

Functionality and the look and feel

There are two parts to this, one is the wearble hand glove that acts like a controller and the other part is the interface where the user can see the immediate effect of their actions with the controller.
