Virtual Piano

A piano that can be played in air

Introduction

What is this about

Virtual piano is exactly what the name suggests, a piano which is intangible but gives a feel of actually playing it. This project is developed as part of a Techno-entertainment show at IIT Madras, to entertain audience with the use of latest technology. The functionality of a piano is mirrored using the X-Box Kinect that detects the hand position with it's motion sensing capabilities. Intuitiveness is added with the help of a dynamically responsive screen that mimics the key presses of an actual piano.

Approach

How is it achieved

Various motion sensing technologies like Proximity sensors, camera, leap motion and kinect were considered but finalized with Kinect as it had depth sensing capabilites and is much accurate than other sensors out there. Hand detection algorithms are used to detect the position of hand with respect to the sensor and notes of the piano are played accordingly. A user interface is designed to add intuitiveness and give an idea of the position of various keys in a piano. The UI is made dynamic with the key presses so as to give enhanced user experience.
